TerminalInfo

class TerminalInfo: NSObject, NSCoding

Info sent via Multipeer connectivity.

Contains:

  • Size of terminal.
  • Message to show in terminal.
  • init(message:terminalSize:)

    Init from given message and size.

    Declaration

    Swift

    Parameters

    message

    Message to show in terminal.
    terminalSize

    Size of terminal (in Floats, not in cols or rows). This is an Array, so provide two values, the first, width and the second, height.
  • message

    Swift

  • terminalSize

    Size of terminal (in Floats, not in cols or rows).

    The first value is the width, and the second value is the height. The setter of this variable put 0 for missing values, or remove extra values.
